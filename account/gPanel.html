    Add Public Key to Authorized Keys for this Account

    How to Generate an SSH Key-Pair
    1. Linux/OSX
      1. Open terminal
      2. Run the command: ssh-keygen -t rsa
      3. Answer the prompts (pressing enter without any input uses default values)
    2. Windows
      1. NEED TO FIGURE THIS OUT USING PuTTYgen
    How to Retrieve a Generated Public Key
    1. Linux
      1. Open terminal
      2. Run the command: cat ~/.ssh/id_rsa.pub
      3. Manually copy the key and paste it into the input field
    2. OSX
      1. Open terminal
      2. Run the command: pbcopy ~/.ssh/id_rsa.pub
      3. Paste the contents into the input field
    3. Windows
      1. NEED TO FIGURE THIS OUT USING PuTTYgen
    How to Access the Server
    1. Linux/OSX
      1. Open terminal
      2. Run the command: ssh account_name@server_name
    2. Windows
      1. Download PuTTY
      2. Open PuTTY
      3. NEEDS REVISION: Input your account name in the username input field
      4. NEEDS REVISION: Input the server address in the server address field
      5. NEEDS REVISION: Go to the SSH tab on the side
      6. NEEDS REVISION: Input your private key
      7. NEEDS REVISION: Click connect
